</p>

### Installation & Usage

```shell 
pip install -r requirements.txt
python script.py
```

### Installation using poetry

If you want to run this script in a virtual environment, you can install poetry `pip install poetry` 

```shell 
poetry install
poetry run python tinder_script.py
# Then you can use the flask app
poetry run python flask_app.py
```
